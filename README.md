# MTR-US-Addon
This mod adds nearly 300 building blocks for American-style stations to Minecraft. Minecraft Transit Railway and Joban Client Mod are required to use this mod. Additionally, the US PIDS Pack resource pack is required to use the PIDS blocks added by this mod.

Currently, only the DC Metro is included, but blocks from other transit systems will be added soon.

**Current block count: 293**
